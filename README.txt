To run this script the following dependencies are needed:
	py -m pip install pillow

This script allows players to play or observe others play minesweeper.

The script is run using:
	py minesweeper_start.py

When run the player chooses whether to play or observe. If the player chooses to
play he can pick the size of the minesweeper game he wants to play. If the player
wants to allow others to observe his gameplay he needs to enter ip and port
information in the form above.
When an observer enters he chooses to observe and need thus to know the ip and port
number of the player server. If the server information is correct the observer will
get the current view of the minesweeper board of the player.
